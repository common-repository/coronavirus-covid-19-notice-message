Simple plugin to quickly display a notice message on your website about the Coronavirus (COVID-19), and include an optional web page for more information.

== Description ==

This plugin adds a Coronavirus notice message to either the top, bottom, or inside the text on a website. This free plugin allows website owners to quickly and easily add a notification banner about COVID-19. Key features include, options for where to display notification (top, top static, bottom, bottom static, or inside your text using a shortcode). Additionally, the plugin offers a optional "read more" link to direct viewers to a web page with more information.

For more information about this plugin you can checkout our web page at [Coronavirus Notice Message plugin](https://www.gallagherwebsitedesign.com/plugin/coronavirus-covid-19-notice-message/).

== Installation ==

The following will guide you through installing and setting up this plugin.

1. Install the plugin by either by uploading it via FTP, or by adding using the "Add Plugin" link when logged into WordPress as an admin.
2. Activate the plugin via the Plugin page when logged into WordPress as an admin.
3. Open up the settings page for the plugn. This can be found in the left hand menu titled "Coronavirus Notice Message"
4. Check the checkbox for " Enable Coronavirus..." this will enable the notification to show on your website.
5. Select the location you want the notice to display. If you choose "inline" then you'll need to copy & paste the shortcode into a web page's content. The shortcode to copy & paste is [coronavirus_notice]
6. Choose the colors you want for the notice background and text.
7. Under the "Content" section add in your notice headline.
8. Type in a notice message in the textarea box.
9. Then optionally, if you want a link to a web page to read more, then enter read more link text.
10. And then select what page/post you want the link to go to.
11. Lastly, click the "Save Changes" button.
12. You're all set! Your notice will now show on your website.

== Frequently Asked Questions ==

= Why should I use this =

You should use this plugin if you want a simple way to tell your visitors how you're handling the Coronavirus as it relates to your business or website. 

= Why is my notice not displaying? =

If your notice is not displaying, make sure you've enabled the message inside the plugin's settings page. Also you may want to try a different message location. The coronavirus banner message location may not be compatible with your theme.

= Is there any cost to use this? =

No, this is a free plugin. It's our way of helping out with the pandemic.
